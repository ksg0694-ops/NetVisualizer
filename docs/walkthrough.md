# 버그 수정 및 UI/UX 구조 개편 완료

요청하신 모든 사항에 대한 수정 및 신규 기능 개발이 완료되었습니다!

## 1. 부동산 청약 탭 구조 최적화 및 대출 시뮬레이션
- **레이아웃 개선:** 3기 신도시 요약 리스트와 수도권 주요 청약 지도를 한 행(Row)에 나란히 배치하여 화면 공간 활용도를 극대화했습니다.
- **예상 대출 금액 기능:** 청약 자금 준비도 카드 바로 옆에 새로운 카드를 추가했습니다. 현재 보유한 현금+안전자산을 기준으로 **50%의 대출이 가능하다고 가정**하여, '예상 대출 가능 금액'과 '총 가용 자금(내 돈 + 대출)'을 실시간으로 계산해서 보여줍니다.

## 2. 포트폴리오 '투자자산 상세 페이지' 및 아코디언 버그 수정
- 포트폴리오 탭에서 스크롤을 내려 **[투자 자산]** (보라색) 항목에 생긴 **`상세보기 >`** 버튼을 눌렀을 때, 아코디언 레이아웃이 깨지던 버그(HTML 버튼 중첩 문제)를 완벽히 수정했습니다.
- 이제 부드럽게 아코디언이 열리고 닫히며, 버튼 클릭 시 별도의 상세 페이지로 쾌적하게 이동합니다.

## 3. 대시보드 구조 원복 및 공백 카드 추가
- 통합된 수입/지출 카드가 너무 크게 늘어나지 않도록 대시보드의 그리드를 다시 기존의 3열(3-cols) 구조로 원복했습니다.
- 기존 지출 카드가 있던 마지막 3번째 열 자리는 비율을 맞추기 위한 **투명한 공란 카드**로 처리하여, PC 화면에서도 깔끔하고 안정적인 비율을 유지하도록 개선했습니다.
